Estrutura de uma página HTML

Esta é a estrutura básica de uma página HTML. Ela inclui as seguintes partes:

  • <!DOCTYPE html>: Declaração do tipo de documento, indicando que é um documento HTML5.
  • <html>: Elemento raiz que envolve todo o conteúdo da página.
  • <head>: Contém metadados, links para arquivos CSS, scripts e o título da página.
  • <body>: Contém o conteúdo visível da página, como texto, imagens e outros elementos.
